🔴 Arduino LED Blinking

Project Overview
This project demonstrates basic Arduino digital output control by blinking an LED at a fixed time interval.  
It is a beginner-level embedded systems project.

Components Used
- Arduino Uno
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

Circuit Description
The LED anode (long leg) is connected to digital pin 3 of the Arduino through a 220Ω current limiting resistor.  
The LED cathode (short leg) is connected to GND.

Working Principle
Arduino sets digital pin 3 HIGH to turn ON the LED and LOW to turn it OFF.  
A delay of 1 second is used between ON and OFF states.
